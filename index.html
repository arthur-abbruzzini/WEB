O que é?

Conceito de web?

Web é uma palavra inglesa que significa teia ou rede. O significado de web ganhou outro sentido com o aparecimento da internet. A web passou a designar a rede que conecta computadores por todo mundo, a World Wide Web (WWW).



O que é o www?

O domínio de site é aquele que aparece logo após o “www” na barra de endereços do seu navegador. Ele é um item básico na criação de um site profissional para sua empresa. A URL do site é um endereço eletrônico ou o caminho pelo qual os usuários chegam até sua marca. Ou seja, conecta seu negócio com o mundo



O que é o w3C?

O que é o W3C? World Wide Web Consortium é uma organização internacional formada por empresas privadas, órgãos governamentais e associações independentes. Os membros da W3C atuam no desenvolvimento dos protocolos e modelos de padronização da rede mundial de computadores (www).



O que é um servidor?

Em informática, um servidor é um software ou computador, com sistema de computação centralizada que fornece serviços a uma rede de computadores, chamada de cliente. Esses serviços podem ser de naturezas distintas, como por exemplo, arquivos e correio eletrônico
