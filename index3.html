Redes

O que é uma LAN?

Uma rede local (LAN) conecta dispositivos que estão fisicamente próximos uns dos outros usando conectores como roteadores e switches. Ela permite que os dispositivos troquem dados e se comuniquem com segurança em pequena escala.



O que é uma MAN?

Uma MAN é maior do que uma LAN, que geralmente engloba um prédio ou um campus, mas é menor do que uma rede de longa distância (WAN), que normalmente engloba uma área geográfica maior.



O que é uma WAN?

Uma rede de longa distância, também chamada de rede de área ampla, rede de grande extensão ou rede de grande alcance é uma rede de computadores que abrange uma grande área geográfica, com frequência um país ou continente.



O que é uma PAN?

(PAN) conecta dispositivos eletrônicos dentro da área imediata de um usuário. O tamanho de uma PAN varia de alguns centímetros a alguns metros. Um dos exemplos mais comuns do mundo real de uma PAN é a conexão entre um fone de ouvido Bluetooth e um smartphone.
